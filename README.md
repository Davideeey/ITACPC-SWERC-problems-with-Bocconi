# ITACPC-SWERC-problems-with-Bocconi

This repository collects part of my solutions in Python to some ICPC-style problems that I worked on during
internal selections, practice sessions, and contests with Bocconi.

## Contents

- `bocconi_icpc_selection_2025-09-20/` – Bocconi internal ICPC selection (Kattis), on 20 Sept 2025, where solving 4 problems ranks second.
- `itacpc_2025/` – ITACPC contest 2025, where solving 3 problems earns a bronze medal
- `practice_swerc_2022_for_swerc_2025/` – practice session using the SWERC 2022 problemset for SWERC 2025.
- `swerc_2025/` – SWERC 2025 contest, where solving 7 problems ranks top 30 pencentile overall, and 7th among italian teams.
